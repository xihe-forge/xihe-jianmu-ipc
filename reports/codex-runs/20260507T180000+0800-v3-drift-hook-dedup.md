## Fix

- Added direct IPC dedup before `Send-IpcWarning`.
- Dedup tuple: `(source IPC_NAME, ai_ts, tool)`.
- Window: 300 seconds by default; override via `TIMESTAMP_DRIFT_WARNING_DEDUP_WINDOW_SECONDS`.
- State: `~/.claude/jianmu-ipc-hooks/dedup-cache.json`; tests use `TIMESTAMP_DRIFT_WARNING_DEDUP_CACHE_PATH`.
- Write path: named mutex + temp JSON + `File.Replace`/`Move-Item` fallback.
- Duplicate behavior: warning/governance audit still writes; harness IPC is suppressed and an `ipc-dedup-suppressed` audit row records key/window/last_sent/age.
- Drift growth alone does not resend IPC inside the window.
