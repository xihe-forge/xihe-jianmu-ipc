# Jianmu usage proxy + claude-hud priority patch

Date: 2026-05-05

## Scope

- Added Jianmu hub `GET /usage` endpoint backed by a central Anthropic OAuth usage proxy.
- Added success cache TTL of 5 minutes and failure cache TTL of 30 seconds.
- Added promise-lock behavior so concurrent cache misses share one Anthropic request.
- Added `patches/claude-hud-jianmu-priority.patch` for `~/.claude/plugins/cache/claude-hud/claude-hud/0.0.7`.

## HUD Upgrade Note

After reinstalling or upgrading claude-hud, apply:

```powershell
cd $env:USERPROFILE\.claude\plugins\cache\claude-hud\claude-hud\0.0.7
git apply D:\workspace\ai\research\xiheAi\xihe-jianmu-ipc\patches\claude-hud-jianmu-priority.patch
npm ci
npm run build
```

The patch keeps the existing Anthropic direct path for hub-down/network/timeout fallback.
