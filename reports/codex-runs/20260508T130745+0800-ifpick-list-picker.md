## 行为

- **数据源**：Hub `/sessions-history?limit=50` + `~/.claude/projects/<encoded-cwd>/*.jsonl` + `~/.codex/sessions/**/*.jsonl` + `~/.codex/archived_sessions/*.jsonl`·四源合并去重
- **排序**：以最新活跃时间排（hub `lastSeenAt` 优先·其次文件 `LastWriteTime`）
- **展示**：`[#] runtime  ipc-name  last-seen  cwd-tail  sessionId-prefix`
- **dispatch**：
  - claude → `ipc <name> -resume <sessionId>`
  - codex 有 IPC name → `ipcx <name> -resume <sessionId>`
  - orphan codex（无 IPC name） → `codex resume <sessionId> --dangerously-bypass-approvals-and-sandbox -c "model_reasoning_effort=\"xhigh\""`
- **dry-run**：env `IFPICK_DRYRUN=1` 仅打印 dispatch 不实际起 session
- **Hub 离线**：自动 fall through 到文件系统源·不报错
